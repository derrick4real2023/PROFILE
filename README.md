# GitHub 个人展示站

本仓库包含一个示例网站，用于在 GitHub Pages 上展示个人项目和学术成果。你可以根据自身需求修改 `index.html` 中的内容。

## 使用方法

1. 在 GitHub 上创建仓库并上传本代码。
2. 打开仓库设置（Settings）中的 **Pages** 选项，选择 `main` 分支作为发布源，目录保持 `/ (root)`。
3. 保存设置后稍等片刻，即可在 `https://你的用户名.github.io/仓库名/` 访问该网站。

## 自定义

- 修改 `index.html` 中的“关于我”“项目列表”和“学术成果”等部分，替换为你的真实信息。
- 若需更多页面或样式，可继续创建新的 HTML、CSS 或 JavaScript 文件。
